SOBRE ESTA FERRAMENTA

Insta Report é uma ferramenta que relata uma conta do instagram direcionada.
Ele usa multi threading, portanto, relata muito rápido.
Esta ferramenta é fortemente recomendada para fins educacionais.


COMO USAR?

Segue os passos para usares a ferramenta : 

WINDOWS

1. Abre o cmd pelo directorio do InstaReport
2. Instala o python 3
3. Escreve "pip install -r requirements.txt" no cmd
4. Escreve "python reportbot.pyc"
